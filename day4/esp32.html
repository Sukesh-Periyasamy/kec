üîå Peripherals

  • CAN bus support
  • 10 capacitive touch sensors

9Ô∏è‚É£ Important Concepts You Must Know

üîπ Flash vs RAM

  • Flash ‚Üí Stores your program (non-volatile, persists after power off)
  • RAM (SRAM) ‚Üí Temporary data while running (lost on power off)

Flash = Hard Drive  |  RAM = Working Memory

üîπ Partition Scheme

Defines how flash memory is divided:

  • App partition ‚Üí Your program code
  • OTA partition ‚Üí Over-The-Air update space
  • SPIFFS / LittleFS ‚Üí File storage
  • NVS ‚Üí Non-volatile key-value storage

üîπ Pull-up / Pull-down Resistors

Used for stable digital input reading:

  • Pull-up ‚Üí Default HIGH, button pulls LOW
  • Pull-down ‚Üí Default LOW, button pulls HIGH
  • ESP32 has internal pull-up/pull-down ‚Äî use INPUT_PULLUP

üîπ Debouncing

Important for button input:

  • Mechanical buttons "bounce" ‚Äî give multiple signals on one press
  • Software debounce: add small delay (~50ms) after button read
  • Hardware debounce: use capacitor across button
