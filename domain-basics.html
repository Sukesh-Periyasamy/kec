PART 1 ‚Äì Basic Concepts (Foundation First)

What is an IP Address?

IP = Internet Protocol Address

An IP address is the real address of a server on the internet.

Example:

142.250.183.14
  • Computers understand IP addresses
  • Humans don't remember them easily
  • So we use domain names instead

What is a Domain Name?

A domain name is a human-readable name for an IP address.

Examples:

  • google.com
  • iitj.ac.in
  • github.com

Instead of typing:

142.250.183.14

We type:

google.com

What is DNS?

DNS = Domain Name System

DNS converts:

google.com ‚Üí 142.250.183.14

Think Like This:

Real Life Internet
Contact Name Domain
Phone Number IP Address
Phonebook DNS

DNS is like the internet's phonebook.

PART 6 ‚Äì How to Connect Custom Domain to GitHub Pages

Now comes the practical teaching part üî•

STEP 1

Deploy Website to GitHub Pages

  1. Push portfolio to GitHub
  2. Go to: Repository ‚Üí Settings ‚Üí Pages
  3. Select: Branch: main
  4. Click Save

Now GitHub gives you a link like:

https://username.github.io/repository-name
STEP 2

Add Custom Domain in GitHub

  1. Go to: Repository ‚Üí Settings ‚Üí Pages
  2. Under: Custom Domain
  3. Enter: yourdomain.com
  4. Click Save

GitHub will automatically create a file called: CNAME in your repository.

STEP 3

Go to Domain Provider DNS Settings

Example in GoDaddy / Namecheap:

  1. Open: DNS Management
STEP 4

Add A Record

Add 4 A Records pointing to GitHub IP:

185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153

Settings:

  • Type: A
  • Host: @
  • Points to: Above IPs (one record for each)
STEP 5

Add CNAME Record

If using www version:

  • Type: CNAME
  • Host: www
  • Points to: username.github.io
STEP 6

Wait for DNS Propagation

Time: 10 minutes ‚Äì 24 hours

After propagation, your site works as:

yourdomain.com
STEP 7

Enable HTTPS

In GitHub Pages settings:

‚úî Enable "Enforce HTTPS"

Now your website is secure!

Common Problems & Solutions

Website Not Opening

Possible Reasons:

  • DNS propagation delay (wait longer)
  • Wrong IP address in A record
  • CNAME typo

HTTPS Not Working

Solutions:

  • Wait a few minutes after enabling
  • Re-check domain spelling in GitHub settings
  • Ensure DNS propagation is complete

Only www Works, Not Root Domain

Solution:

  • Check A record is properly configured with @ as host
  • Ensure all 4 GitHub IPs are added
