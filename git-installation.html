STEP 1

Download Git

Official Website:

https://git-scm.com/downloads

Git automatically detects your operating system.

For Windows:

Click: Download for Windows

This will download a .exe installer file.

STEP 2

Install Git on Windows

  1. Double-click the .exe file
  2. Click Yes (if User Account Control appears)
  3. Click Next on most screens (default settings are recommended)

Important Installation Settings

Pay close attention during installation. Students will see multiple configuration options.

STEP 3

Verify Git Installation

Now we check whether Git is installed correctly.

Method 1 ‚Äì Using Command Prompt

  1. Press Windows + R
  2. Type: cmd
  3. Press Enter
  4. Type: git --version

If installed correctly, output will be like:

git version 2.45.1.windows.1

Method 2 ‚Äì Using PowerShell

  1. Open PowerShell
  2. Run: git --version

Method 3 ‚Äì Using Git Bash

  1. Right-click on Desktop
  2. Click Git Bash Here
  3. Type: git --version

If You See Version ‚Üí Installation Successful! üéâ

Common Errors & Fixes

Error 1: "git is not recognized as an internal or external command"

Reason: Git was not added to PATH.

Solution:

  1. Reinstall Git
  2. Make sure you select: "Git from the command line and also from 3rd-party software"
  3. OR manually add Git to Environment Variables

Error 2: Antivirus blocking installation

Solution: Disable antivirus temporarily and install again.

Error 3: Installed but still not working

Solution: Restart your system.

Sometimes PATH updates require a system restart.

STEP 4

Configure Git (Very Important After Installation)

Every student must run these commands after installation:

git config --global user.name "Your Full Name"
git config --global user.email "your@email.com"

Example:

git config --global user.name "Sukesh Periyasamy"
git config --global user.email "m24im1007@iitj.ac.in"

Verify Configuration

Run:

git config --list

It should show your name and email.

What to Tell Students

Explain clearly:

  • Git is not just a tool.
  • It is a developer's identity system.
  • Every commit is recorded with your name and email.
  • This information appears in the project history forever.
