What Goes Inside Each File?

index.html

Contains:

  • Website structure
  • Sections
  • Text content
  • Links to CSS & JS

Basic structure:

<!DOCTYPE html>
<html>
<head>
  <title>My Portfolio</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <section id="hero">
    <h1>Your Name</h1>
    <p>Web Developer</p>
  </section>

  <script src="script.js"></script>
</body>
</html>

Explain:

  • <head> ‚Üí metadata
  • <body> ‚Üí visible content
  • <link> ‚Üí connects CSS
  • <script> ‚Üí connects JS

style.css

Controls:

  • Colors
  • Layout
  • Fonts
  • Spacing
  • Responsive design

Example:

body {
  font-family: Arial;
  margin: 0;
}

#hero {
  text-align: center;
  padding: 50px;
}

script.js

Controls:

  • Button clicks
  • Animations
  • Menu toggle
  • Smooth scrolling

Example:

console.log("Portfolio Loaded");

PHASE 2 ‚Äì Using Portfolio Templates (Real Industry Workflow)

After they understand structure, now move to:

https://themewagon.com/theme-tag/portfolio-template/

Explain:

These are pre-designed themes made by professional designers.

How to Download Template

  1. Choose a template
  2. Click Download
  3. Extract ZIP file
  4. Open folder in VS Code

Understanding Template Structure

Template usually contains:

index.html
css/
js/
images/
fonts/

Explain:

  • css/ folder ‚Üí multiple style files
  • js/ folder ‚Üí animation & effects
  • images/ ‚Üí all graphics
  • fonts/ ‚Üí custom typography

Important Teaching Point

Tell students:

Don't change design randomly.
First understand how sections are structured.

How to Customize Template

STEP 1

Change Text

  1. Open index.html
  2. Find: John Doe
  3. Replace with: Your Name
STEP 2

Replace Images

  1. Go to: images/ folder
  2. Replace: profile.jpg
  3. Keep same file name or update path
STEP 3

Edit Skills

Find skill section in HTML.

Modify:

<li>HTML</li>
<li>CSS</li>
STEP 4

Add Projects

  1. Duplicate project card block
  2. Update: Title
  3. Update: Description
  4. Update: GitHub link
STEP 5

Update Contact Section

Add:

  • Email
  • LinkedIn link
  • GitHub link

PHASE 3 ‚Äì Connect to GitHub

After customization:

Open terminal inside project folder:

git init
git add .
git commit -m "Portfolio customized"
git remote add origin https://github.com/username/portfolio.git
git push -u origin main

Enable GitHub Pages.

Now portfolio is live!
