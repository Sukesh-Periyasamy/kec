03 Â· Process

ğŸŒ How to Access a Website?

Every time you open a web page, a 5-step process happens in milliseconds. Hover each step to see the details.

1

User Enters URL

The user types the website address (URL) into the browser â€” e.g. https://example.com

2

Browser Sends Request

The browser contacts a DNS server to resolve the domain name into an IP address, then sends an HTTP/HTTPS request to the web server.

3

Server Processes Request

The web server receives the request. For static sites, it retrieves HTML files. For dynamic sites, it runs server-side code and queries databases.

4

Server Sends Response

The server sends back the HTML, CSS, JavaScript, and other assets as an HTTP response to the browser.

5

Browser Renders Page

The browser parses the HTML, applies CSS styles, executes JavaScript, and displays the final visual web page to the user.
