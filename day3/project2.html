Sections 3 & 4

üåç What is an API & API Key?

An API (Application Programming Interface) lets two systems communicate. Our React app talks to the WeatherAPI server and gets live data back.

üì°

Our App = Frontend

We send a request: "Give me weather for Chennai"

üåê

WeatherAPI = Backend

They respond with structured JSON weather data.

üì¶

JSON Response

We parse the data and display it as cards, charts, and icons in our UI.

Sample API response

{ "temp_c": 32, "humidity": 60, "wind_kph": 15, "condition": { "text": "Partly Cloudy" } }

üîë What is an API Key?

üé´

Unique Secret Code

Identifies your specific application to the WeatherAPI servers.

üìà

Tracks Usage

API providers monitor how many requests your key makes per day.

üõ°Ô∏è

Prevents Misuse

Without a key, anyone could spam the server ‚Äî keys add access control.

# Example API request URL https://api.weatherapi.com/v1/forecast.json?key=YOUR_KEY&q=Chennai&days=7
‚ö†Ô∏è Never expose API keys publicly. Do not paste them in your HTML/JS files directly. Use .env files instead.

üîê What is a .env File?

.env stores sensitive variables outside your code so they are never pushed to GitHub.

# .env file VITE_WEATHER_API_KEY=your_secret_key_here
// In React (Vite) ‚Äî access the key securely const API_KEY = import.meta.env.VITE_WEATHER_API_KEY;
üí° Add .env to your .gitignore file so it is never uploaded to GitHub.

üå¶ Weather API Fields Explained

WeatherAPI returns rich JSON data. Here are all the fields we use across different parts of our dashboard.

‚òÅÔ∏è Current Weather Fields

Field Why We Use It
temp_c Main temperature display on the hero card
condition.icon Animated weather icon (sunny, rainy, cloudy etc.)
condition.text Human-readable condition description
humidity Moisture level shown as circular progress bar
wind_kph Wind speed shown in bar chart
feelslike_c "Feels like" temperature card
uv UV index warning card
pressure_mb Atmospheric pressure stat card
vis_km Visibility distance card

üìÖ 7-Day Forecast Fields

Field Why We Use It

avgtemp_c Used in the temperature line chart
totalprecip_mm Total rainfall amount
daily_chance_of_rain Rain probability badge on forecast card
maxwind_kph Wind trend across the week
condition.icon Weather icon for each forecast day

Section 14

üì± Mobile-First Design Guidelines

A weather app is primarily used on mobile. Design for small screens first, then scale up for desktop.

üì¶

Flexbox / Grid

Use CSS Flexbox for card stacking and Grid for the forecast layout on desktop.

üìê

Stack Vertically

On mobile, all cards should stack in a single column for easy scrolling.

üëÜ

Large Touch Buttons

Minimum 44px touch targets for search, refresh, and forecast day buttons.

üìä

Responsive Charts

Recharts has a ResponsiveContainer ‚Äî always wrap charts with it for mobile.

üóÇÔ∏è

Collapsible Sections

Let users expand/collapse hourly forecast to save screen space on mobile.

üß≠

Bottom Navigation

Modern mobile apps use bottom nav bars for switching between views (Today, Week, Map).
