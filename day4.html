What is Embedded System?

Definition

An embedded system is a small computer designed to perform a specific task.

Examples

  • Washing machine controller
  • Smartwatch
  • ECG monitor
  • Smart irrigation system

Core Components

  • Microcontroller / Microprocessor
  • Sensors
  • Power supply
  • Communication interface

What is a Microcontroller?

A microcontroller is a small computer on a single chip.

It Includes:

  • CPU
  • RAM
  • Flash memory
  • GPIO pins
  • Communication modules

Examples:

  • Arduino Uno
  • ESP32

What is ESP32?

Definition:

ESP32 is a powerful microcontroller with:

  • Built-in WiFi
  • Built-in Bluetooth
  • Dual-core processor
  • Multiple GPIO pins

Why ESP32?

  • No need for external WiFi module
  • Cheap and powerful
  • Ideal for IoT projects
  • Works with Arduino IDE

ESP32 Important Pins

3.3V

Power output pin

GND

Ground pin

GPIO

General purpose I/O

TX/RX

Serial communication

VIN

Voltage input

What is Arduino?

Definition:

Arduino is an easy-to-use microcontroller platform.

Popular Board: Arduino Uno

Features:

  • ATmega328P microcontroller
  • USB programming
  • 14 Digital Pins
  • 6 Analog Pins
  • 5V logic

When to Use Arduino?

  • Basic projects
  • Learning electronics
  • When WiFi is not required

What is Raspberry Pi?

Definition:

Raspberry Pi is a small computer.

Key Difference:

  • Arduino/ESP32 = Microcontroller
  • Raspberry Pi = Microcomputer

It Can:

  • Run Linux
  • Connect keyboard and monitor
  • Run Python
  • Host web servers

Use Cases:

  • AI projects
  • Server hosting
  • Image processing

Microcontroller vs Microcomputer

Feature Arduino / ESP32 Raspberry Pi

Communication Protocols

Protocols are rules for communication between devices.

I2C

Inter-Integrated Circuit

SPI

Serial Peripheral Interface

UART

Universal Asynchronous Receiver/Transmitter

I2C (Inter-Integrated Circuit)

Features:

  • 2 wires (SDA, SCL)
  • Master-Slave architecture
  • Address based

Used in:

  • OLED display
  • RTC modules

SPI (Serial Peripheral Interface)

Features:

  • 4 wires (MISO, MOSI, SCK, CS)
  • Faster than I2C
  • Short distance

Used in:

  • SD card module
  • TFT display

UART (Universal Asynchronous Receiver/Transmitter)

Features:

  • TX and RX pins
  • Serial communication
  • Used for programming

Example:

  • USB to Serial communication

Complete ESP32 Code

Replace:

  • YOUR_WIFI
  • YOUR_PASSWORD
  • YOUR_WRITE_API_KEY
#include <WiFi.h>
#include <HTTPClient.h>
#include "DHT.h"

#define DHTPIN 4
#define DHTTYPE DHT11

const char* ssid = "YOUR_WIFI";
const char* password = "YOUR_PASSWORD";

String apiKey = "YOUR_WRITE_API_KEY";

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(115200);
  dht.begin();

  WiFi.begin(ssid, password);
  Serial.print("Connecting");

  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }

  Serial.println("\nConnected to WiFi");
}

void loop() {
  if (WiFi.status() == WL_CONNECTED) {

    float temp = dht.readTemperature();
    float hum = dht.readHumidity();

    if (!isnan(temp) && !isnan(hum)) {

      HTTPClient http;

      String url = "http://api.thingspeak.com/update?api_key=" 
                   + apiKey + "&field1=" + String(temp) 
                   + "&field2=" + String(hum);

      http.begin(url);
      int httpCode = http.GET();
      http.end();

      Serial.println("Data sent to ThingSpeak");
    }
  }

  delay(15000);
}
