What is an Operating System?

OS = Bridge between User & Hardware

  • üëâ Manages hardware & software resources
  • üëâ Provides User Interface (UI) for you

Examples:

Windows, Linux, macOS, Android, iOS

Core Responsibilities of OS

  • Process Management: Running and scheduling programs
  • Memory Management: Allocating and managing RAM
  • File System: Organizing and storing data
  • Device Drivers: Communicating with hardware
  • Security: Managing user accounts and permissions

Without an OS?

  • ‚ùå No application can run
  • ‚ùå No memory management possible
  • ‚ùå No file access or storage
  • ‚ùå No intuitive user interface

OS is mandatory for system operation.

Kernel vs User Mode

User Mode

  • Applications run here
  • Limited hardware access
  • Safe: Crash = App closes
VS

Kernel Mode

  • Full hardware access
  • Core system control
  • Critical: Crash = Blue Screen
Why separation? ‚Üí To ensure Security & Stability.

What is Virtualization?

Virtualization = Running multiple Operating Systems on one physical machine simultaneously.

Example setup:

One Physical Laptop ‚Üí
Windows (Host) + Linux (Guest) + Kali (Guest)

What is a Hypervisor?

Hypervisor = Software that creates & manages Virtual Machines.

  • Allocates CPU, RAM, & Disk space
  • Controls VM execution like a conductor

Type 1 (Bare Metal)

Runs directly on hardware.
Used in Enterprise Severs.

Type 2 (Hosted)

Runs as an app on OS.
VirtualBox, VMware Workstation.

üß† Quick Quiz: Hypervisor

What software manages and runs Virtual Machines?

A) Kernel
B) Hypervisor
C) Bootloader
D) Device Driver

‚úÖ (B) Hypervisor

A hypervisor (like VirtualBox) acts as the platform that creates and runs virtual machines on top of the physical hardware.

üß† Quick Quiz: Kernel Mode

Which mode provides full, unrestricted access to hardware?

A) User Mode
B) Safe Mode
C) Kernel Mode
D) Dark Mode

‚úÖ (C) Kernel Mode

Kernel Mode is the privileged ring of the CPU where the OS core runs. Restricting apps to User Mode prevents them from crashing the whole system.

üß† Quick Quiz: Snapshots

Why is taking a Snapshot important?

A) Takes a photo of you
B) Speeds up the Internet
C) Saves exact system state

‚úÖ (C) Saves exact system state

Snapshots are your "Undo" button. They allow you to revert the VM back to a previous healthy state if an installation fails.

üß† Quick Quiz: Network Modes

Which network mode gives the VM its own unique IP from the router?

A) NAT
B) Bridged Adapter
C) Host-Only

‚úÖ (B) Bridged Adapter

Bridged mode connects the VM directly to your physical network, making it look like a distinct device separate from your host computer.
