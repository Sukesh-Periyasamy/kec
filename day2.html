Virtualization Technology

Running Computers Inside Computers

Virtualization allows you to run multiple Operating Systems simultaneously on a single physical machine using a Hypervisor.

Type 1 Hypervisor

Bare Metal

  • Installs directly on hardware
  • Used in Enterprise Servers
  • Examples: VMware ESXi, Microsoft Hyper-V

Type 2 Hypervisor

Hosted (Our Lab)

  • Installs as an app on OS
  • Used for Testing/Learning
  • Examples: VirtualBox, VMware Workstation

‚ö†Ô∏è The "PC Can't Run Windows 11" Fix (TPM Bypass)

At the installation setup screen, press Shift + F10 to open CMD.

  1. Type regedit and press Enter.
  2. Navigate to: HKEY_LOCAL_MACHINE\SYSTEM\Setup
  3. Right-click Setup -> New Key -> Name it LabConfig
  4. Inside LabConfig, create 2 DWORDs (Value = 1):
    • BypassTPMCheck
    • BypassSecureBootCheck

Network Modes Explained

NAT

Safe & Simple. VM shares host IP. Good for web browsing.

Bridged Adapter

Pro Mode. VM gets own IP from router. Acts as separate physical device.

Host-Only

Isolated. Private network between Host and VM. No Internet.
