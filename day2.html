What is an Operating System?

OS = Bridge between User & Hardware

  • üëâ Manages hardware & software resources
  • üëâ Provides User Interface (UI)

Examples:

Windows, Linux, macOS, Android

Core Responsibilities of OS

  • Process Management: Running and scheduling programs
  • Memory Management: Allocating and managing RAM
  • File System: Organizing and storing data
  • Device Drivers: Communicating with hardware
  • Security: Managing user accounts and permissions

  • ‚ùå No application can run

What is Virtualization?

Virtualization = Running multiple OS on one physical machine.

Example:

One Laptop ‚Üí Windows (Host) + Linux (Guest) + Kali (Guest)

What is a Hypervisor?

Hypervisor = Software that creates & manages VMs

  • Allocates CPU, RAM, Disk
  • Controls VM execution

Type 1

Runs on Bare Metal (Servers)

Type 2

üß† Quick Quiz: Hypervisor

What software manages Virtual Machines?

(A) Kernel
(B) Hypervisor
(C) Bootloader
(D) Driver

‚úÖ (B) Hypervisor

A hypervisor (like VirtualBox) is responsible for creating and running virtual machines, allocating resources like CPU and RAM.

üß† Quick Quiz: Kernel Mode

Which mode has full access to hardware?

(A) User Mode
(B) Safe Mode
(C) Kernel Mode
(D) Dark Mode

‚úÖ (C) Kernel Mode

Kernel mode is the core of the OS with unrestricted access to system memory and hardware. User applications run in User Mode for safety.

üß† Quick Quiz: Snapshots

Why is a Snapshot important?

(A) Takes a photo of the user
(B) Speeds up the internet
(C) Saves system state to revert changes

‚úÖ (C) Saves system state to revert changes

Snapshots allow you to save the exact state of a VM. If you break something, you can restore the snapshot instantly.

üß† Quick Quiz: Network Modes

Which network mode makes the VM appear as a separate device on your WiFi?

(A) NAT
(B) Bridged Adapter
(C) Host-Only

‚úÖ (B) Bridged Adapter

Bridged mode connects the VM directly to the physical network, giving it its own IP address from the router, just like a separate physical computer.
