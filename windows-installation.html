STEP 1

üîê Enable Virtualization

Before installing:

  1. Press Ctrl + Shift + Esc
  2. Go to Performance ‚Üí CPU
  3. Check: Virtualization

If it shows Disabled:

  1. Restart computer
  2. Enter BIOS (F2 / F10 / DEL depending on system)
  3. Enable:
    • Intel VT-x
    • SVM Mode (AMD)
  4. Save & Exit
STEP 2

üß∞ Install VirtualBox

  1. Open downloaded VirtualBox file
  2. Click Next ‚Üí Next ‚Üí Install
  3. Allow network driver installation
  4. Click Finish
  5. Restart system if required
STEP 3

üñ•Ô∏è Create Windows 11 Virtual Machine

1. Open VirtualBox

Click "New"

2. Fill details

  • Name: Windows 11
  • Type: Microsoft Windows
  • Version: Windows 11 (64-bit)

Click Next

3. Allocate RAM

  • Minimum: 4GB
  • Recommended: 8GB
Do not give more than half your total RAM

Click Next

4. Allocate CPU

Go to Processor tab:

  • Minimum: 2 cores
  • Recommended: 4 cores

Click OK

5. Create Virtual Hard Disk

Select:

  • Create virtual hard disk now
  • VDI
  • Dynamically allocated
  • Size: 80GB

Click Finish

STEP 5

üìÄ Attach Windows ISO

  4. Click OK
STEP 6

üöÄ Install Windows 11

  1. Click Start
  2. Select Language
  3. Click Install Now
  4. If asked for product key:
    • Enter key OR
    • Click "I don't have a product key"
  5. Choose version: Windows 11 Pro

‚ö†Ô∏è IMPORTANT: Fix "This PC Can't Run Windows 11"

If you see this error:

2. Type:

3. Navigate to:

4. Create new key:

  • Right click ‚Üí New ‚Üí Key
  • Name it: LabConfig

5. Inside LabConfig create 3 DWORD (32-bit) values:

6. Close Registry Editor

‚ùå Black Screen After Boot

‚ùå Very Slow Performance

‚ùå No Internet in VM
